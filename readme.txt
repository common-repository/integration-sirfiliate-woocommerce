Launch and manage your affiliate program, connect with the Sir Filiate platform and automate your affiliate processes.



== Description ==

Easily integrate Sir Filiate with your WooCommerce store and launch your affiliate program in no time!

Sir Filiate enables you to launch your affiliate program and start growing your business. Create, manage and pay affiliates seamlessly and fully automated with just a few clicks.
Invoices are automatically created and shared with your affiliates to save you as much time as possible, leaving you more time to focus on your business.
The Sir FIliate integration is compatible with WooCommerce and WooCommerce Subscriptions.

Key Features:
-> Automatic Integration with WooCommerce (API Key)
-> Set a minimum order value to assign commission
-> Set the cookie lifetime
-> Disable individual products from commission
-> Automated payments and Invoicing
-> Assign order to affiliate manually



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/integration-sirfiliate-woocommerce` directory, or install the plugin through the WordPress plugins section directly.
1. Activate the plugin through the 'Plugins' section in WordPress
1. Go to WooCommerce->Settings->Sir Filiate to configure the plugin

== Frequently Asked Questions ==

= How do I connect the plugin with the Sir Filiate platform? =

Create an API key in the Sir Filiate platform and paste it into the plugin, click and save and authorize. The plugin will be connected.

= Do I need to purchase a plan at Sir Filiate? =

Yes, the plugin is offered for free as integration but the platform should be purchases via https://sirfiliate.com

= Which languages are supported? =

The platform is available in Dutch and English

= What automated payment solutions does Sir Filiate offer? =

The Sir Filiate automated payments are based on a bulk SEPA batch file.
